# 文字RPG - 冒險之旅

一個網頁版的文字類型RPG遊戲，包含行動、打怪、打Boss等核心功能。

## 如何運行

### 方法一：直接打開（最簡單）
1. 雙擊 `index.html` 文件
2. 遊戲會在瀏覽器中打開

### 方法二：使用本地服務器（推薦）
1. 雙擊 `start_server.bat` 啟動服務器
2. 在瀏覽器中訪問：`http://localhost:8000`
3. 按 `Ctrl+C` 可以停止服務器

### 方法三：使用Python命令
```bash
python -m http.server 8000
```
然後在瀏覽器中訪問：`http://localhost:8000`

## 遊戲功能

- **角色系統**：生命值、等級、經驗值、攻擊力、防禦力、金幣
- **探索系統**：在不同地點探索，可能遇到怪物或找到金幣
- **戰鬥系統**：與怪物和Boss戰鬥
- **商店系統**：購買道具增強實力
- **升級系統**：獲得經驗值升級，提升屬性

## 遊戲地點

1. **起始村莊** - 安全的地點，適合休息
2. **黑暗森林** - 可能遇到普通怪物
3. **荒蕪平原** - 可能遇到普通怪物
4. **惡魔洞穴** - Boss的巢穴，可以挑戰Boss

## 操作說明

- **探索**：在當前地點探索，可能遇到怪物或找到金幣
- **戰鬥**：與遇到的敵人戰鬥
- **挑戰Boss**：在惡魔洞穴中挑戰強大的Boss
- **休息**：恢復生命值
- **商店**：購買各種道具
- **移動**：選擇不同地點進行冒險

想要永久部署遊戲並讓朋友也能訪問？請查看 **[DEPLOY.md](DEPLOY.md)** 詳細部署指南！

### 快速部署選項：
1. **GitHub Pages** - 最推薦，完全免費，永久可用
2. **Netlify** - 最簡單，拖放即可部署
3. **Vercel** - 快速部署，自動HTTPS

部署後你會得到一個網址，例如：
- `https://你的用戶名.github.io/text-rpg-game` (GitHub Pages)
- `https://你的遊戲名稱.netlify.app` (Netlify)
